# Radiant Protocol Tutorial – Run It Yourself

This tutorial will walk you through running the entire Radiant stack locally using Docker, then deploying to Sepolia testnet.

## Prerequisites
- Docker and Docker Compose
- Node.js 18+ (for Hardhat)
- MetaMask browser extension
- Some Sepolia testnet ETH (get from a faucet)
