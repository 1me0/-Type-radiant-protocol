# Radiant Protocol Tutorial – Run It Yourself

This tutorial will guide you through setting up and running the entire Radiant Protocol stack locally using Docker, deploying smart contracts to a testnet, and interacting with the system.

## Prerequisites

- **Docker** and **Docker Compose** (version 2.0+)
- **Node.js** 18+ (for Hardhat)
- **MetaMask** browser extension
- **Sepolia testnet ETH** – obtain from a faucet (e.g., [Sepolia Faucet](https://sepoliafaucet.com/))
- **Git** (to clone the repository)
