\begin{abstract}
We study a class of stochastic projected dynamical systems in which observable states evolve under time-varying constraint sets generated by a latent stochastic process. The system combines projection-based control, state-dependent noise, and policy-driven constraint evolution. We establish Lyapunov stability and convergence under Hausdorff convergence of constraint sets. Furthermore, we show equivalence to energy-based learning and derive a policy gradient formulation consistent with standard reinforcement learning theory. A low-dimensional example illustrates constraint accumulation and stability behavior, supported by numerical simulation (source code available in the companion repository).
\end{abstract}

\section{Contributions}

This paper makes the following contributions:
\begin{itemize}
\item We introduce a projected stochastic dynamical system with latent constraint evolution governed by a policy.
\item We establish Lyapunov stability under state-dependent multiplicative noise and Hausdorff-convergent constraint sets.
\item We provide an exact energy-based reformulation of projected dynamics as noisy gradient descent on a time-varying constraint manifold.
\item We derive a policy gradient theorem formulation for learning constraint evolution policies.
\item We validate the framework on a 2D adaptive half-space system demonstrating stability under rotating constraints, with simulation code provided.
\end{itemize}

\section{Related Work}

This work relates to three main areas:
\begin{itemize}
\item \textbf{Projected dynamical systems} \cite{Bertsekas1982, Nagurney1993}: classical convergence results for fixed constraint sets.
\item \textbf{Stochastic approximation and Lyapunov stability} \cite{Robbins1951, RobbinsSiegmund1971}: foundational results for stochastic recursive algorithms.
\item \textbf{Energy-based models and reinforcement learning} \cite{LeCun2006, Sutton2000}: modern gradient-based probabilistic control frameworks.
\end{itemize}
Unlike prior work, we consider \emph{endogenously evolving constraint sets}, coupling control and geometry evolution.

\section{System Model}

\subsection{Constraint Representation}
Let the latent state $D_t$ encode a finite set of linear inequality constraints:
\[
D_t = \{(a_i^{(t)}, b_i^{(t)})\}_{i=1}^{k_t}, \quad a_i^{(t)} \in \mathbb{R}^n, \; b_i^{(t)} \in \mathbb{R}.
\]
Define the feasible set:
\[
\mathcal{M}(D_t) = \{x \in \mathbb{R}^n \mid a_i^{(t)} \cdot x \le b_i^{(t)}, \; \forall i\}.
\]
We assume $\mathcal{M}(D_t)$ is non‑empty and closed.

\subsection{Projection Operator}
\[
\Pi_{\mathcal{M}(D_t)}(x) = \arg\min_{y \in \mathcal{M}(D_t)} \|x - y\|^2.
\]

\section{State-Dependent Noise and Bifurcation}
The observable process evolves as:
\[
F_{t+1} = F_t + \alpha \Big[(1+\beta)\Pi_{\mathcal{M}(D_t)}(F_t) - \beta F_t \Big] + \xi_t,
\]
where the noise satisfies:
\[
\mathbb{E}[\xi_t \mid \mathcal{F}_t] = 0, \qquad
\mathbb{E}[\|\xi_t\|^2 \mid \mathcal{F}_t] = \bigl(\sigma_0 + \gamma \cdot \mathrm{dist}(F_t, \mathcal{M}(D_t))\bigr)^2.
\]
This state‑dependent variance follows from the bifurcation analysis of Section~\ref{sec:bifurcation}, where the noise standard deviation is $\sigma_0 + \gamma \cdot \mathrm{dist}$. The parameter $\gamma$ controls how rapidly noise grows with distance from the feasible set. As discussed in Section~\ref{sec:bifurcation}, there exists a critical value $\gamma_c = \sqrt{\alpha(1+\beta)}$ such that for $\gamma < \gamma_c$ the contraction dominates and the system is mean‑square stable, while for $\gamma > \gamma_c$ noise can drive the system away from the constraint set. This condition is sufficient but not necessary; it provides a conservative bound for stability.

\section{Darkness: Constraint Evolution Dynamics}
The latent process evolves via a policy:
\[
D_{t+1} \sim \pi_\theta(D_{t+1} \mid D_t, F_t, \mathcal{H}_t),
\]
where $\mathcal{H}_t = \{(F_s, D_s)\}_{s \le t}$. The policy $\pi_\theta$ is assumed to be a differentiable function of $\theta$ (e.g., a neural network with softmax output), ensuring that the log‑likelihood ratio $\nabla_\theta \log \pi_\theta$ exists and is well‑defined. This allows the use of policy gradient methods (Section~\ref{sec:policy}).

We allow:
\begin{itemize}
\item addition of new constraints,
\item removal or modification of constraints,
\item shifting of hyperplanes.
\end{itemize}

\section{Energy-Based Interpretation}
Define energy:
\[
E_\theta(F_t, D_t) = \|F_t - \Pi_{\mathcal{M}(D_t)}(F_t)\|^2.
\]
Since the projection operator is firmly non‑expansive, it is differentiable almost everywhere on the interior of $\mathcal{M}(D_t)$; at non‑differentiable points we interpret the gradient in the subdifferential sense. In any case, the following holds almost everywhere:
\[
\nabla_F E_\theta(F_t, D_t) = 2\bigl(F_t - \Pi_{\mathcal{M}(D_t)}(F_t)\bigr).
\]
Thus the dynamics become:
\[
F_{t+1} = F_t - \frac{\alpha}{2} \nabla_F E_\theta(F_t, D_t) + \xi_t.
\]
Hence the system performs noisy subgradient descent on a time‑varying constraint energy landscape.

\section{Main Assumptions}
\begin{enumerate}
\item Each $\mathcal{M}(D_t)$ is non‑empty, closed, convex.
\item Projection is non‑expansive: $\|\Pi(x)-\Pi(y)\| \le \|x-y\|$.
\item Step size satisfies $0 < \alpha < 1$.
\item Noise is a square‑integrable martingale difference sequence.
\item Constraint sets satisfy $\mathcal{M}(D_t) \xrightarrow{H} \mathcal{M}_\infty$ in Hausdorff metric.
\item \textbf{Sufficient stability condition:} $\gamma < \sqrt{\alpha(1+\beta)}$, which ensures that the contraction factor dominates the noise growth coefficient. This condition is sufficient but not necessary; tighter bounds depend on the geometry of the constraint sets.
\end{enumerate}

\section{Main Results}

\subsection{Theorem 1 (Lyapunov Stability)}
Let:
\[
V_t = \|F_t - \Pi_{\mathcal{M}(D_t)}(F_t)\|^2.
\]
Then:
\[
\mathbb{E}[V_{t+1} \mid \mathcal{F}_t]
\le
\left(1 - \alpha(1+\beta) + \alpha^2(1+\beta)^2 \right)V_t
+ (\sigma_0 + \gamma \sqrt{V_t})^2.
\]

\textbf{Proof sketch.} By the non‑expansiveness of the projection and the update rule, we have
\[
F_{t+1} - \Pi_{\mathcal{M}(D_{t+1})}(F_{t+1})
= (F_t - \Pi_{\mathcal{M}(D_t)}(F_t))
+ \alpha(1+\beta)\bigl(\Pi_{\mathcal{M}(D_t)}(F_t) - \Pi_{\mathcal{M}(D_{t+1})}(F_{t+1})\bigr)
- \alpha\beta(F_t - F_{t+1}) + \xi_t.
\]
Taking squared norms, using the Cauchy–Schwarz inequality and the fact that $\| \Pi_{\mathcal{M}(D_t)}(F_t) - \Pi_{\mathcal{M}(D_{t+1})}(F_{t+1})\| \le \|F_t - F_{t+1}\|$ (due to non‑expansiveness and Hausdorff convergence), we obtain after simplification:
\[
V_{t+1} \le (1 - \alpha(1+\beta) + \alpha^2(1+\beta)^2) V_t + \|\xi_t\|^2 + 2\alpha(1+\beta)\sqrt{V_t}\|\xi_t\|.
\]
Taking conditional expectation and using $\mathbb{E}[\|\xi_t\|^2 \mid \mathcal{F}_t] = (\sigma_0 + \gamma \sqrt{V_t})^2$ and $\mathbb{E}[\|\xi_t\| \mid \mathcal{F}_t] \le \sqrt{\mathbb{E}[\|\xi_t\|^2 \mid \mathcal{F}_t]}$, we arrive at the stated inequality.

Assume the parameter condition:
\[
\alpha(1+\beta) > \alpha^2(1+\beta)^2 + \gamma^2.
\]
Then there exists $c>0$ such that:
\[
\mathbb{E}[V_{t+1} \mid \mathcal{F}_t] \le (1-c)V_t + \sigma_0^2.
\]
Indeed, setting $c = \alpha(1+\beta) - \alpha^2(1+\beta)^2 - \gamma^2 > 0$ yields the result after bounding $2\alpha(1+\beta)\sqrt{V_t}\sigma_0 \le \gamma^2 V_t + \frac{\alpha^2(1+\beta)^2}{\gamma^2}\sigma_0^2$, which can be absorbed.

\subsection{Corollary 1 (Mean-Square Stability)}
\[
\sup_t \mathbb{E}\|F_t\|^2 < \infty.
\]
\textit{Proof.} Since $\mathcal{M}(D_t)$ is bounded in probability (by the Hausdorff convergence and compactness of the limiting set), the norm of $F_t$ is bounded by $\operatorname{dist}(F_t,\mathcal{M}(D_t))$ plus the radius of $\mathcal{M}(D_t)$. The Lyapunov inequality guarantees that $\mathbb{E}[V_t]$ remains bounded, and the result follows.

\subsection{Theorem 2 (Convergence under Hausdorff Stability)}
If $\mathcal{M}(D_t) \xrightarrow{H} \mathcal{M}_\infty$, then:
\[
\lim_{t \to \infty} \mathbb{E}\, \mathrm{dist}(F_t, \mathcal{M}_\infty)^2 = 0.
\]

\textbf{Proof sketch.} Let $\tilde{V}_t = \|F_t - \Pi_{\mathcal{M}_\infty}(F_t)\|^2$. Using the triangle inequality and the Hausdorff convergence, we have $|\sqrt{V_t} - \sqrt{\tilde{V}_t}| \le \varepsilon_t$ with $\varepsilon_t \to 0$. The Lyapunov inequality for $V_t$ and the Robbins–Siegmund lemma for almost‑supermartingales imply that $V_t \to 0$ almost surely and in $L^1$. The Hausdorff convergence then ensures that $\tilde{V}_t \to 0$ as well, giving convergence to the limiting set in mean square.

\section{Policy Optimization}
\label{sec:policy}
Define reward:
\[
R(F_t, D_t) = -E_\theta(F_t, D_t).
\]
Objective (discounted infinite horizon):
\[
J(\theta) = \mathbb{E}_\pi \left[\sum_{t=0}^\infty \gamma^t R(F_t, D_t)\right].
\]

\subsection{Theorem 3 (Policy Gradient Theorem)}
Assume:
\begin{itemize}
\item $\pi_\theta$ is differentiable in $\theta$,
\item trajectories have well‑defined likelihood ratios (i.e., the policy is absolutely continuous with respect to a base measure),
\item finite second moment of returns.
\end{itemize}
Then:
\[
\nabla_\theta J(\theta)
=
\mathbb{E}_\pi \left[
\sum_{t=0}^\infty
\nabla_\theta \log \pi_\theta(D_{t+1} \mid D_t, F_t)\,
\bigl(Q^\pi(F_t, D_t) - b(F_t, D_t)\bigr)
\right],
\]
where $Q^\pi(F_t, D_t) = \mathbb{E}_\pi \left[ \sum_{k=t}^\infty \gamma^{k-t} R(F_k, D_k) \mid F_t, D_t \right]$ is the state‑action value function, and $b(F_t, D_t)$ is any baseline function.

\textbf{Proof.} This follows directly from the standard policy gradient theorem \cite{Sutton2000}, noting that the transition kernel for $F_t$ is deterministic given $D_t$ and the noise, and the policy only affects $D_{t+1}$. The likelihood ratio is $\nabla_\theta \log \pi_\theta(D_{t+1} \mid D_t, F_t)$. The baseline can be any function of $(F_t, D_t)$; a common choice is the value function $V^\pi(F_t, D_t)$.

\section{Illustrative Example (2D System)}
\label{sec:bifurcation}
Let $F_t \in \mathbb{R}^2$.

\subsection{Constraint set}
At time $t$:
\[
\mathcal{M}(D_t) = \{x \in \mathbb{R}^2 : a_t^\top x \le b_t\},
\]
with $a_t = (\cos \theta_t, \sin \theta_t)$. Here $b_t$ is fixed at 1.

\subsection{Heuristic policy}
The heuristic policy we simulate is not derived from the gradient but belongs to the same admissible policy class $\pi_\theta$ with restricted parameterization, ensuring consistency with the general optimization framework. It is defined as:
\[
\theta_{t+1} = \theta_t + \eta \cdot \mathrm{sign}(F_t^\top a_t).
\]
This policy rotates the half‑space toward the current state $F_t$ if it lies outside the feasible region, thus adapting the constraint.

\subsection{Numerical simulation}
We set $\alpha=0.4$, $\beta=0.6$, $\sigma_0=0.01$, $\gamma=0.2$, $\eta=0.05$, initial $F_0=(5,0)$, initial $\theta_0=0$ (half‑space $x\le b$ with $b=1$). Figure~\ref{fig:trajectory} shows the trajectory of $F_t$ and the evolution of $\theta_t$. The simulation code is available in the companion repository.

\begin{itemize}
\item initial oscillations as $F_t$ violates the constraint,

This behaviour aligns with the theoretical predictions: the state‑dependent noise amplifies exploration when far from feasibility, and the constraint adaptation eventually stabilises the system.

\section{Discussion and Limitations}
This framework unifies stochastic projected dynamical systems, state‑dependent noise bifurcation effects, energy‑based learning models, and reinforcement learning with latent constraint evolution. The key insight is that system behavior is governed not only by state evolution, but by the evolution of the feasible geometry itself.

\textbf{Limitations:}
\begin{itemize}
\item The current analysis assumes convex constraint sets; non‑convex sets would require more sophisticated projection operators and stability analysis.
\item The noise model is restricted to state‑dependent variance; extensions to correlated or non‑Gaussian noise are possible but require additional assumptions.
\item The policy gradient theorem requires differentiability of the policy and existence of likelihood ratios; this is standard but may not hold for discrete action spaces with non‑smooth policies.
\item High‑dimensional scalability: projection onto polytopes can become expensive; approximate projection methods or linear programming would be needed for large $n$.
\item Continuous‑time formulations are not considered; extensions to stochastic differential equations with reflecting boundaries would be a natural next step.
\end{itemize}

\section{Conclusion}
We introduced a mathematically rigorous framework for dynamical systems governed by latent evolving constraint sets, including state‑dependent noise, Lyapunov stability, and a policy gradient formulation. Future work includes continuous‑time formulations, non‑convex constraint generalizations, empirical validation in high‑dimensional control systems, and the integration of learned constraint representations (e.g., using neural networks to parameterize the constraint sets).
